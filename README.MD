
# Exemple de programme pour Gamebuino META

Version 2.0

Ce petit programme permet d'apprécier le même programme :
* [en procédurale](https://github.com/chris-scientist/gb-sample/archive/v1.0.zip) ;
* et [en orienté objet](https://github.com/chris-scientist/gb-sample/releases/tag/v2.0).

Le programme fait bouger un personnage, qui est représenté par un *x*, dans les limites de l'écran.
